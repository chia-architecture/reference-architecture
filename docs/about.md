# About

Caribbean Healthcare Interoperability Architecture is an open and collaborative initiative to define a shared architecture for cross-border healthcare interoperability in the Caribbean.

## Community

This is community-driven project and is intended to bring together healthcare professionals, architects, policymakers, and technologists interested in improving cross-border healthcare in the caribbean.

## Get involved

We welcome contributions from anyone.  
See the [Contribute](contribute.md) page for more information.

## Status

The project is currently in an early stage and is evolving based on feedback and collaboration across the region.
